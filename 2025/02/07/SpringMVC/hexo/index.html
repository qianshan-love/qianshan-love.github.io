1. 打开主题配置文件

找到你的 Fluid 主题配置文件 _config.fluid.yml，通常位于 themes/fluid/_config.yml。

2. 添加自定义 CSS 和 JS

在 _config.fluid.yml 文件中，找到 custom_css 和 custom_js 配置项。如果这两个配置项不存在，你可以手动添加它们。

3. 添加鼠标效果的 CDN 链接

根据你想实现的效果，将对应的 CDN 链接添加到 custom_js 或 custom_css 中。
